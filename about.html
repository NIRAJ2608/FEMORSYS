About FEmoRSys

This application leverages deep learning to detect facial emotions from images and live video streams. It is built using Flask for the interface and utilizes a trained machine learning model for emotion recognition.

📊 Dataset: FER 2013

The Facial Expression Recognition 2013 (FER-2013) dataset contains 35,887 grayscale images of 48x48 pixels, labeled with seven emotions.

😊 The 7 Recognized Emotions

😠 Angry | 🤢 Disgust | 😨 Fear | 😀 Happy | 😢 Sad | 😲 Surprise | 😐 Neutral
